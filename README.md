# wordbook
This is a very simple java web project named "wordbook". 

这是一个简单的单词本项目，主要应用了Java web的相关技术，采用JSP、Servlet搭配MySQL完成。

这是一个非常实用的Java Web练手项目，采用Factory类处理数据库JDBC操作。

**单词管理系统** 主要包含增、删、查、改四部分功能。
